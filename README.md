# Orders Microservice

This is a NestJS microservice for managing orders.

## Description

This microservice is part of the Backend-MS project. It handles creating, reading, updating, and deleting orders. It communicates with other microservices via TCP.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
